# Assets Folder

Upload your images and videos to this folder using the file manager.

## Required Files for the Website

### For the Memories Gallery
Upload 6 images with these exact names:
- memory-1.jpg
- memory-2.jpg
- memory-3.jpg
- memory-4.jpg
- memory-5.jpg
- memory-6.jpg

(You can use .jpg, .jpeg, .png, or .webp formats)

### For the Birthday Video (Optional)
If you want to host the birthday video locally instead of YouTube:
- birthday-video.mp4

## How to Upload Files

1. Click on the "public" folder in the file tree
2. Click on the "assets" folder
3. Click the "Upload" button
4. Select your images/videos
5. Make sure they have the correct filenames as listed above

## Notes

- The website will automatically load these files from /assets/
- Recommended image size: 1920x1080 or smaller
- Recommended image file size: Under 5MB each for best performance
- For videos, consider using YouTube embeds (already configured) for better performance
